# ♻️ RecyConnect

RecyConnect is a full-stack waste management platform designed to encourage sustainable recycling through a Waste-to-Worth reward system. Users can submit recyclable waste, earn Eco-Points based on the type and weight of waste, track their recycling activity, and redeem accumulated points for rewards.

## ✨ Features

- User registration and login
- Secure password hashing using bcrypt
- Waste pickup scheduling
- Automatic Eco-Point calculation
- Personal dashboard with recycling statistics
- Waste submission history
- Eco-Point rewards and redemption system
- RecyBot chatbot for recycling-related queries
- Cloud-hosted PostgreSQL database

## 🛠️ Tech Stack

**Frontend:** HTML5, CSS3, JavaScript  
**Backend:** Node.js, Express.js  
**Database:** PostgreSQL (Neon)  
**Authentication:** bcrypt password hashing  

## 🚀 How It Works

1. Users create an account or log in.
2. Users submit recyclable waste details and schedule a pickup.
3. Eco-Points are calculated based on the waste type and weight.
4. Recycling activity and accumulated points are displayed on the dashboard.
5. Users can redeem their Eco-Points for available rewards.
